# Antibiotics Dosage Calculator

這個計算機會根據多重過濾條件，提供建議的抗生素施打劑量及建議。
[原始資料參考表單](https://docs.google.com/spreadsheets/d/1EuLjXpxsH09rv67WEvhjbKCvarg31_GrjJXeJnY4Ok0/edit?gid=1812450353#gid=1812450353)

## 基本使用流程

- 提供計算CrCl值所需的基本資訊(或是直接提供CrCl)
- 選取當前是否有洗腎(Dialysis)
- 勾選病原菌及抗藥性篩選 (可多選)
- 選取篩選出適合的抗生素種類 (可多選)
- 顯示選取的抗生素建議施打劑量及文字備註

## 細節需求

- 若有選洗腎狀態，則將CrCl計算值的框框改為顯示洗腎狀態的值，並將底色改為黃色
- 參照原表單將抗生素分類，可用底色區分或做成兩階清單
- 紀錄查詢歷史，自動帶入所有欄位的值(cookie? 後端?)
- 要有可以直接輸入CrCl值的地方，參照目前版本的做法
- 輸入框加入驗證/限制
    - 年齡只能輸入正整數
    - 體重 < 30 或 > 150要上色warning
    - Creatine > 10 要上色warning
- RWD要確認手機不會跑版

## 反查抗菌譜

- 獨立開另一個頁面
- 選取多個抗生素
- 根據選擇顯示那些菌株有被cover哪些沒有
